### –†–∞–±–æ—Ç–∞ —Å –ø–æ—Ç–æ–∫ –∑–∞ –≤—Ö–æ–¥ –æ—Ç —Ñ–∞–π–ª ([ifstream](https://en.cppreference.com/w/cpp/io/basic_ifstream))


–ü—Ä–∏–º–µ—Ä –∑–∞ –ø—Ä–æ—á–∏—Ç–∞–Ω–µ –Ω–∞ —Ü—è–ª–æ—Ç–æ —Å—ä–¥—ä—Ä–∂–∞–Ω–∏–µ —Ñ–∞–π–ª:

```c++
		return -1;
	}
```

### –†–∞–±–æ—Ç–∞ —Å –ø–æ—Ç–æ–∫ –∑–∞ –∏–∑—Ö–æ–¥ –∫—ä–º —Ñ–∞–π–ª ([ofstream](https://en.cppreference.com/w/cpp/io/basic_ofstream))

```c++

	std::ofstream file(FILE_NAME);  // create output file stream associated with myFile.txt

		std::cout << "Error!" << std::endl;
		return -1;
	}

	int b = 10;

	file << a << " " << b << " " << a + b << std::endl; // write into the output file stream

		std::cout << "The file contains more data after the two integers!" << std::endl;
	}

	file.close();
}
```
